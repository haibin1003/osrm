# 业务系统管理 (REQ-200~220)

## 文档信息
- **需求编号**: REQ-200 ~ REQ-220
- **模块名称**: 业务系统管理
- **优先级**: P1
- **状态**: 已实现

## 1. 功能概述

业务系统管理用于管理企业内部的各种业务系统，建立业务系统与开源软件的关联关系，支持按业务系统维度统计开源软件使用情况。

## 2. 功能清单

### REQ-200 业务系统基础管理
- 创建、编辑、删除业务系统
- 业务系统编码、名称、描述
- 负责人配置
- 启用/禁用业务系统

### REQ-201 业务域管理
- 创建、编辑、删除业务域
- 业务域分类（电商、金融、政务等）
- 业务系统归属业务域

### REQ-202 业务系统与软件关联
- 业务系统关联已订购的软件
- 查看业务系统的软件使用列表
- 软件使用统计（按业务系统维度）

## 3. API接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 业务域列表 | GET | /api/v1/business/domains | 获取业务域列表 |

| 业务系统列表 | GET | /api/v1/business/systems | 获取业务系统列表 |

| 删除业务系统 | DELETE | /api/v1/business/systems/{id} | 删除业务系统 |
