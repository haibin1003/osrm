# OSRM UI/UX 设计规范

本文档定义 OSRM 系统的视觉设计规范、交互规范和组件使用指南。

---

## 文档信息

- **作者**: OSRM 架构团队
- **创建时间**: 2026-03-18
- **版本**: v1.0

---

## 1. 设计原则

### 1.1 设计目标

- **专业性**: 体现企业级软件的专业、可靠形象
- **高效性**: 帮助用户快速完成任务，减少操作步骤
- **一致性**: 统一的视觉语言和交互模式
- **可访问性**: 符合 WCAG 2.1 AA 标准

### 1.2 设计关键词

```
专业 · 简洁 · 现代 · 高效
```

## 3. 字体系统

### 3.1 字体族

```css
/* 中文优先 */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;
```

### 3.2 字号规范

| 样式 | 大小 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| **Heading 1** | 24px | 600 | 32px | 页面标题 |
| **Heading 2** | 20px | 600 | 28px | 区块标题 |
| **Heading 3** | 16px | 600 | 24px | 卡片标题 |
| **Body** | 14px | 400 | 22px | 正文内容 |
| **Body Small** | 13px | 400 | 20px | 辅助文字 |
| **Caption** | 12px | 400 | 18px | 标签、说明 |

### 3.3 字重规范

| 字重 | 值 | 用途 |
|------|-----|------|
| Regular | 400 | 正文、描述 |
| Medium | 500 | 次级标题、强调 |
| Semibold | 600 | 标题、按钮、重要文字 |

## 11. 响应式适配

### 11.1 移动端适配原则

- 侧边栏默认收起，通过汉堡菜单展开
- 表格横向滚动或使用卡片式布局
- 表单单列布局
- 操作按钮分组或放入更多菜单
