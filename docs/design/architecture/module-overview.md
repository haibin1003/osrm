## 1. 用户中心模块 (User Center)

| 属性 | 内容 |
|------|------|
| **模块编号** | REQ-001 ~ REQ-099 |
| **模块名称** | 用户中心 (User Center) |
| **模块职责** | 负责用户的认证、授权、会话管理和权限控制，是整个系统的安全入口 |

### 核心功能

| 功能编号 | 功能名称 | 功能描述 | 优先级 |
|---------|---------|---------|--------|
| REQ-001-001 | 用户登录 | 用户名密码登录，支持JWT Token | P0 |
| REQ-001-002 | Token刷新 | 使用Refresh Token换取新的Access Token | P0 |
| REQ-001-003 | 用户登出 | 安全退出，吊销Token | P0 |
| REQ-001-004 | 密码管理 | 修改密码、重置密码、密码策略 | P1 |
| REQ-001-005 | 会话管理 | 并发控制、超时登出 | P1 |
| REQ-001-006 | 权限控制 | RBAC权限模型实现 | P0 |

### 输入/输出

| 类型 | 内容 |
|------|------|
| **输入** | 用户名、密码、Token、权限配置 |
| **输出** | JWT Token、用户信息、权限列表、会话状态 |

### 依赖关系

| 类型 | 模块 |
|------|------|
| **依赖模块** | 无（基础模块） |
| **被依赖模块** | 所有其他模块 |

### 关键实体

- User（用户）
- Role（角色）
- Permission（权限）
- LoginLog（登录日志）

### 接口概览

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 登录 | POST | /api/v1/auth/login | 用户登录 |
| 刷新Token | POST | /api/v1/auth/refresh | 刷新Access Token |
| 登出 | POST | /api/v1/auth/logout | 用户登出 |
| 获取当前用户 | GET | /api/v1/auth/me | 获取登录用户信息 |
