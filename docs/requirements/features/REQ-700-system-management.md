# 系统管理 (REQ-700~720)

## 文档信息
- **需求编号**: REQ-700 ~ REQ-720
- **模块名称**: 系统管理
- **优先级**: P1
- **状态**: 已实现

## 1. 功能概述

系统管理提供用户管理、角色管理、权限管理、系统设置等功能，仅限系统管理员操作。

## 2. 功能清单

### REQ-700 用户管理
- 用户列表查询
- 创建用户
- 更新用户信息
- 删除用户
- 重置密码
- 启用/禁用用户

### REQ-701 角色管理
- 角色列表查询
- 创建角色
- 更新角色
- 删除角色
- 启用/禁用角色

### REQ-702 权限管理
- 权限树展示
- 查看权限列表
- 权限与角色关联

### REQ-703 系统设置
- 系统参数配置
- 存量登记功能开关
- 其他系统参数

### REQ-704 使用统计
- 软件包统计
- 订购统计
- 业务系统统计
- 趋势分析

## 3. API接口

### 用户管理
| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 用户列表 | GET | /api/v1/users | 用户列表 |
| 用户详情 | GET | /api/v1/users/{id} | 用户详情 |
| 创建用户 | POST | /api/v1/users | 创建用户 |
| 更新用户 | PUT | /api/v1/users/{id} | 更新用户 |
| 删除用户 | DELETE | /api/v1/users/{id} | 删除用户 |
| 重置密码 | PUT | /api/v1/users/{id}/password | 重置密码 |
| 切换状态 | PUT | /api/v1/users/{id}/status | 启用/禁用 |

### 角色管理
